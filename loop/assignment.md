##Assignment
The objective is to create a Python program that continuously prompts the user to input a word using an infinite while True loop. Once a word is provided, the program must ask the user to enter a single letter to search for within that word. The program must validate this second input: if the user enters more than one character (or zero characters), it should display an error message stating "wrong input , only one ,enter again" and repeatedly prompt them until a single letter is provided. After receiving a valid letter, the program checks if the letter exists in the word. If it does, it outputs "yes found" along with the exact number of times that letter appears in the word. If it is not found, it should inform the user, and then the main loop restarts to ask for a new word.
